Download Seeded

Get the newest release from GitHub. The link below always points to the latest version.

Download latest View all releases

Install (Debian/Ubuntu/ValhallaOS)

sudo apt install ./seeded_<version>_amd64.deb

Verify checksums

If you upload a SHA256SUMS file in your release assets:

sha256sum -c SHA256SUMS

Use immediately

seeded render my.seed -o out --version 1.0 --maintainer "Your Name <you@example.com>"
