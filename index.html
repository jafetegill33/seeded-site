Generate Debian metapackages from .seed files.

Seeded is a fast, minimal CLI written in C that turns simple seed definitions into Debian source package trees. Perfect when you’re building a distro with debootstrap / cdebootstrap and need repeatable package sets.

Download latest Read docs View source

Tip: The “Download latest” button always points to the newest GitHub Release.

Quick commands

seeded verify<file.seed>
seeded render-o out
seeded build-o build

One-liner

seeded render my.seed -o out --version 1.0 --maintainer "Your Name <you@example.com>"

Debian-correct output

Seeded generates a clean debian/ tree you can build with dpkg-buildpackage or sbuild.

Made for distro builders

Great for “desktop profiles”, base systems, branding metapackages, and repeatable installation sets.

Fast + minimal

Native C tooling designed to fit into CI/build farms and custom repositories like VAK.
