Seeded Docs

Seeded generates Debian metapackage source trees from .seed files. Common workflow: write seed → render → build with Debian tools.

Commands

seeded verify <file.seed>

seeded verify path/to/package.seed

seeded print-deps <file.seed>

seeded print-deps path/to/package.seed

seeded render <file.seed> -o <outdir> [options]

seeded render package.seed -o out \ --version 1.0 \ --maintainer "Your Name <you@example.com>" \ --distro valhalla

seeded build <file.seed> -o <outdir> [options]

seeded build package.seed -o build \ --version 1.0 \ --maintainer "Your Name <you@example.com>"

Seed file example

Package: my-distro-desktop Section: metapackages Priority: optional Description: My distro desktop A desktop metapackage generated from a seed file. Depends: xfce4 lightdm network-manager

Generated output

<package>/ ├── debian/ │ ├── control │ ├── rules │ ├── changelog │ ├── copyright │ └── source/format └── README.seeded
